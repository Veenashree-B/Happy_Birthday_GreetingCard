# Happy Birthday Greeting Card App 🎉

A visually appealing Android application designed to serve as a beautiful digital birthday greeting card. This project demonstrates foundational skills in modern Android UI development, focusing on layout composition, resource management, and delivering a delightful user experience.

## ✨ Features
* **Elegant UI Design:** Features a warm, inviting layout using high-quality image resources and beautifully styled typography.
* **Declarative UI:** Built entirely with **Jetpack Compose**, showcasing a modern approach to structuring Android application interfaces.
* **Efficient Resource Management:** Demonstrates the proper integration and scaling of image drawables and string resources within the Compose ecosystem.
* **Responsive Layout:** Elegantly handles text and image alignment to ensure a great visual experience on various Android devices.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Design System:** Material Design 3
* **Build System:** Gradle

## 🚀 Getting Started

To run this project locally on your machine:

1. Clone this repository to your local machine:
   ```bash
   git clone [https://github.com/Veena-shree/happy_birthday_greetingcard.git](https://github.com/Veena-shree/happy_birthday_greetingcard.git)
Open the project in Android Studio.

Allow Gradle to sync and download the necessary dependencies.

Click the Run button to launch the app on an Android Emulator or a physical device.
